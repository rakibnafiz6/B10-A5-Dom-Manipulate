Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model is a programming interface for web documents.It represent the page so that programs can change the document structure, style, and content.

How do you select an element from the dom?

1.Using getElementById: The getElementById method allows you to select an element by its ID. 2.Using getElementsByClassName, 3.Using getElementByTagName, 4.Using querySelector and 5.Using querySelectorAll etc.

What is event delegation in the context of the dom, and why it is useful?

The process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.It allows us to attach a single event listener for elements that exist now or in the future.

How do you manipulate an element's attributes and styles using the Dom?

If you want to change other attributes like id, href, or src, you can use the setAttribute() method. The setAttribute() method accepts two arguments: The name of the attribute to set.The value of the attribute to set.
